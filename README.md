Car Rental System

A web-based Car Rental System designed to simplify vehicle booking and management. The system provides an efficient platform for users to rent vehicles while allowing administrators to manage cars, bookings, and user accounts in a centralized system.

Features
User authentication (login and registration)
Vehicle management (add, update, and delete vehicles)
Booking system for renting vehicles
Admin dashboard for system management
Organized data handling for transactions and records
Technologies Used
Java Spring Boot (Backend)
MySQL (Database)
HTML / CSS (Frontend)
Maven (Build and dependency management)
How to Run the Project

Create the database:
- CREATE DATABASE car_rental_db;
Ensure that the MySQL server is running.

Run the Spring Boot application using Maven:
- mvn spring-boot:run

Open the application in your browser:
- http://localhost:8080

Project Overview
  - This system simulates a real-world car rental workflow, allowing users to browse and book vehicles while enabling administrators to manage inventory and monitor transactions efficiently.
